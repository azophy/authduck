Authduck is an OpenID Connect server playground built for development or learning. You could use this website or install it yourself.

To start playing with this tools, you could use any OIDC client you have (for example: openidconnect.net , an OIDC client playground by Auth0), and then use the /.well-known/openid-configuration path to populate all the required fields.
